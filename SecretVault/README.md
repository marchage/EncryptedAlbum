# Secret Vault - Clean & Simple

A beautiful macOS app for password-protected photo vaults.

## Quick Start

1. **Open Xcode**
2. **File → New → Project → macOS App**
3. Product Name: **SecretVault**
4. **Save it** (anywhere is fine)
5. In Xcode, **delete** the default files:
   - Delete `SecretVaultApp.swift`
   - Delete `ContentView.swift`
6. **Add** all 6 Swift files from this SecretVault folder:
   - Right-click project → Add Files to "SecretVault"...
   - Select all `.swift` files
   - Check "Copy items if needed"
7. Press **⌘R** to run

## Features

✅ Create multiple vaults
✅ Each vault has its own password
✅ Beautiful SwiftUI interface
✅ Lock/unlock vaults
✅ Ready to add photo import & encryption

## What You'll See

1. Welcome screen with "Create New Vault" button
2. Create vault (choose name, color, password)
3. Unlock vault with password
4. Empty vault ready for photos

Clean, simple, and it works!
